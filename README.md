Finance-CampBX
==============

Access to the CampBX bitcoin trading API


INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

This module requires these other modules and libraries:

  LWP::UserAgent;
  JSON
